程序流程结构

概述

C 语言支持最基本的三种程序运行结构：顺序结构、选择结构、循环结构。

  • 顺序结构：程序按顺序执行，不发生跳转
  • 选择结构：依据是否满足条件，有选择执行相应功能。(if、if else、if elseif else、switch case)
  • 循环结构：条件是否满足，循环多次执行某段代码。(while、do while、for、)

选择结构：

  • if是一个关键字，if后面有()，if()后面是没有分号
  • ()里写的是条件，条件为真，执行if语句，为假不执行
  • if()后面最好加{},{}代表if语句
  • 如果不加，只有第一语句属于if
  • if()后面没有分号
  • if else结构，else不是必须的，可以不写，else后不能接条件，else if可以接条件
  • 三目运算符就是简洁版的if else
  • switch是关键字，switch()后面没有分号
  • switch()中()里只能放整型变量或一个字符型变量
  • switch()类似于电梯模型（开关），每个case必须有break，跳出switch语句
  • switch效率相对会比if效率高

  • while循环：判断()内条件，如果为真，执行{}语句，如果为假，跳出while循环，执行完{}语句再重复上次步骤。
  • do while循环：do while第一次无条件执行，第二次开始，和while一样。

跳转语句(break、continue、goto)

break语句：

在switch语句和循环语句中都可以使用break语句。

  • 当它出现在switch语句中，作用是终止某个case并跳出switch结构
  • 当它出现在循环语句中，作用是跳出当前内循环语句，执行后面的代码
  • 当它出现在嵌套循环语句中，跳出最近的内循环语句，执行后面的代码
  • 易错提示：break只能跳出循环语句和switch，不能跳出if语句

continue语句:

在循环语句中，跳过一轮循环，继续往下一轮执行

goto语句:

goto语句在任意地方都可以使用，无条件跳转，尽量少用，不要滥用，只能跳转到同一个作用域{}中
