函数的分类

C程序由函数组成，我们写代码都是由主函数main()开始执行的。函数是C程序的基本模块，适用于完成特定任务的程序代码单元。
从函数定义的角度，函数可分为系统函数和用户定义函数两种：

  • 系统函数，即库函数：这是由编译系统提供，用户不必自己定义这些函数，可以直接使用他们，如常用的打印函数printf()
  • 用户定义函数：专门解决用户特点需求

函数的作用

程序就像是一个公司，而函数就是公司中的每个部门，公司规模更加清晰，高效。

  • 函数的使用可以省去重复的代码编写，降低代码重复率
  • 函数可以让程序更加模块化，从而有利于程序的阅读，修改和完善
  1. 函数的定义，就是将main()函数内部代码换个地方
  2. 函数的定义，类似厨房做好的菜，没人点就没有意义

函数的定义，只有用户调用才起到作用！

如何调用函数库(别人写好的函数)

  1. 头文件：包含指定的头文件，头文件主要包含此函数的声明
  2. 函数名字：函数名字必须和头文件声明的一样
  3. 功能：需要知道此函数能干嘛后才调用
  4. 参数：参数类型要匹配
  5. 返回值：根据需要接收返回值，也就是函数的返回值可以不用

自定义函数

无参无返回值

无参无返回值的定义：

  1. 没有返回值，用void关键字修饰
  2. 没有参数，也可用void，或者空
  3. 第一个void代表返回值，function代表函数名，()内部代表参数，空就是无参，{}就是函数体
  4. 同一个文件不能出现同名函数
  5. 不同函数，内部变量是没有关系

有参无返回值

有参无返回值的定义：

  1. 定义函数的参数叫做形参，形参的格式必须为：类型+变量，不能赋值
  2. 函数没有调用，形参变量不会分配空间，函数调用完毕，形参变量自动释放
  3. 如果形参有多个，用逗号”,”分割

无参有返回值

无参有返回值函数定义

  1. 第一个int代表函数的返回类型，函数内部通过return返回
  2. return作用：终止当前的函数
  3. 返回可以是常量、变量、表达式
  4. 函数返回值只能返回一个，不能返回多个
  5. 如果return的类型和函数返回类型不一致，以函数返回类型为准

有参有返回值

有参有返回值函数定义

  1. 定义函数的参数叫做形参，形参的格式必须为：类型+变量，不能赋值
  2. 函数没有调用，形参变量不会分配空间，函数调用完毕，形参变量自动释放
  3. 如果形参有多个，用逗号”,”分割
  4. 第一个int代表函数的返回类型，函数内部通过return返回
  5. 返回可以是常量、变量、表达式

return函数和exit函数的作用

  1. return主函数main()，程序结束
  2. return其他函数，程序不结束
  3. exit()结束整个程序，结束进程

函数的声明和定义

  1. 编译器很笨，在mian函数调用其它函数，只会往前找函数的定义
  2. 如果找不到定义，那么找函数声明，如果没有找到，C编译器会有警告但可以运行，C++会出错
  3. 如果没有定义，直接调用一个不存在的函数会报错
  4. 函数的声明，告诉编译器，这个函数是有定义的，只是放在别的地方
  5. 如果定义没有放在main函数的前面，那么调用函数前需要加声明，声明加不加extern是一样的效果
  6. 一个函数只能定义一次，但是可以多次声明
  7. 声明的形参变量名和定义的变量名可以不一样
  8. 可以只声明，不定义函数，此函数不能调用（无意义）
  9. 声明时，形参变量名可以不写,可以只写类型，定义时一定要写

分文件（多文件）

  1. 按功能划分 xxx.h xxx.c
    • xxx.c 函数的定义
    • xxx.h 函数的声明
  2. 有一个主函数测试自定义函数 main.c
  3. 编译：gcc *.c -o test
  4. 多个文件中，不能出现同名函数（static除外）

注意：

  • 为啥不能在头文件中定义函数：当引用该文件的时候宏替换，会将函数定义都展开到引用文件造成体积庞大代码冗余，并且多文件引用时会造成重复定义错误
